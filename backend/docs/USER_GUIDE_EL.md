# 📱 AURA - Οδηγός Χρήστη (Ελληνικά)

Καλώς ήρθατε στον AURA! Αυτός είναι ο πλήρης οδηγός χρήσης της πλατφόρμας.

---

## 🚀 Γρήγορη Έναρξη

### 1. Εγγραφή

1. Ανοίξτε την εφαρμογή AURA
2. Πατήστε "Εγγραφή"
3. Συμπληρώστε:
   - Email
   - Κωδικός πρόσβασης (τουλάχιστον 8 χαρακτήρες)
   - Όνομα
4. Πατήστε "Εγγραφή"

### 2. Σύνδεση

1. Εισάγετε το email και τον κωδικό σας
2. Πατήστε "Σύνδεση"
3. Αν έχετε ενεργοποιήσει 2FA, εισάγετε τον κωδικό από την εφαρμογή authenticator

### 3. Ενεργοποίηση 2FA (Συνιστάται)

1. Μεταβείτε στις Ρυθμίσεις
2. Επιλέξτε "Ασφάλεια"
3. Πατήστε "Ενεργοποίηση 2FA"
4. Σαρώστε το QR code με την εφαρμογή authenticator (Google Authenticator, Authy, κλπ)
5. Εισάγετε τον πρώτο κωδικό για επιβεβαίωση
6. Αποθηκεύστε τους backup codes σε ασφαλές μέρος

---

## 💰 Χρήση της Πλατφόρμας

### Προβολή Assets

1. Μεταβείτε στην καρτέλα "Assets"
2. Δείτε όλα τα διαθέσιμα assets (μετοχές, κρύπτο, πολύτιμα μέταλλα)
3. Πατήστε σε ένα asset για λεπτομέρειες

### Προβλέψεις Τιμών

1. Επιλέξτε ένα asset
2. Πατήστε "Προβλέψεις"
3. Δείτε τις προβλέψεις για:
   - 30 λεπτά
   - 1 ώρα
   - 24 ώρες
4. Δείτε το confidence score και το sentiment analysis

### Paper Trading

1. Μεταβείτε στην καρτέλα "Paper Trading"
2. Επιλέξτε asset
3. Πατήστε "Αγορά" ή "Πώληση"
4. Εισάγετε ποσότητα και τιμή
5. Επιβεβαιώστε την συναλλαγή

### Portfolio

1. Μεταβείτε στην καρτέλα "Portfolio"
2. Δείτε:
   - Τρέχουσες θέσεις
   - Κέρδη/Ζημίες (P&L)
   - Σύνοψη portfolio
   - Ιστορικό συναλλαγών

## 🔒 Ασφάλεια

### Κωδικός Πρόσβασης

- Χρησιμοποιήστε ισχυρό κωδικό (τουλάχιστον 8 χαρακτήρες)
- Μην τον μοιράζεστε με κανέναν
- Αλλάξτε τον τακτικά

### 2FA

- Συνιστάται η ενεργοποίηση 2FA
- Χρησιμοποιήστε backup codes σε περίπτωση απώλειας συσκευής
- Κρατήστε την εφαρμογή authenticator ασφαλή

### API Keys

- Αν συνδέσετε brokers, κρατήστε τα API keys ασφαλή
- Μην τα μοιράζετε ποτέ
- Χρησιμοποιήστε read-only keys όταν είναι δυνατό

## ❓ Συχνές Ερωτήσεις

### Πώς λειτουργούν οι προβλέψεις;

Οι προβλέψεις βασίζονται σε Machine Learning models που αναλύουν:
- Ιστορικά δεδομένα τιμών
- Νέα και sentiment
- Τεχνικά indicators
- Macro οικονομικά δεδομένα

### Είναι ασφαλές το Paper Trading;

Ναι! Το Paper Trading είναι 100% εικονικό. Δεν χρησιμοποιεί πραγματικά χρήματα.

### Πώς λειτουργεί το Live Trading;

Για Live Trading, πρέπει να συνδέσετε τα API keys από broker (π.χ. Binance, eToro). Το AURA εκτελεί trades μέσω αυτών των keys, αλλά δεν έχει πρόσβαση στα χρήματά σας.

### Μπορώ να αλλάξω τον κωδικό μου;

Ναι, μεταβείτε στις Ρυθμίσεις > Ασφάλεια > Αλλαγή Κωδικού.

## 🆘 Αντιμετώπιση Προβλημάτων

### Δεν μπορώ να συνδεθώ

1. Ελέγξτε ότι το email και ο κωδικός είναι σωστά
2. Αν έχετε 2FA, ελέγξτε τον κωδικό
3. Χρησιμοποιήστε backup code αν χάσατε πρόσβαση

### Οι προβλέψεις δεν φαίνονται

1. Ελέγξτε τη σύνδεση internet
2. Περιμένετε λίγα δευτερόλεπτα
3. Κάντε refresh την σελίδα

### Σφάλμα σύνδεσης

1. Ελέγξτε τη σύνδεση internet
2. Ελέγξτε αν ο server είναι online
3. Επικοινωνήστε με support
